# User Stories - Smart Clinic Management System

### 1. Doctor User Story: View Patient Medical History
**Title:** Access Patient Records  
**Criteria:** As a Doctor, I want to view a patient's historical medical records and previous prescriptions so that I can provide an informed diagnosis during the appointment.  
**Priority:** High  
**Story Points:** 5

### 2. Patient User Story: Book Appointment
**Title:** Appointment Scheduling  
**Criteria:** As a Patient, I want to browse available time slots for a specific doctor and book an appointment online so that I can visit the clinic without waiting.  
**Priority:** High  
**Story Points:** 8

### 3. Admin User Story: Manage Doctor Profiles
**Title:** Doctor Onboarding  
**Criteria:** As an Admin, I want to add new doctors to the system and update their specialities or availability so that the clinic directory remains accurate.  
**Priority:** Medium  
**Story Points:** 3

### 4. Doctor User Story: Update Availability
**Title:** Manage Daily Slots  
**Criteria:** As a Doctor, I want to update my available working hours for the week so that patients do not book appointments when I am unavailable.  
**Priority:** High  
**Story Points:** 5

### 5. Patient User Story: Cancel Appointment
**Title:** Appointment Cancellation  
**Criteria:** As a Patient, I want to cancel my existing appointment at least 24 hours in advance so that the time slot becomes available for other patients.  
**Priority:** Medium  
**Story Points:** 3
