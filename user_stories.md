# User Stories - Smart Clinic Management System

### 1. Doctor User Story
**Title:** View Patient History  
**Criteria:** As a Doctor, I want to view a patient's medical history before an appointment.  
**Priority:** High  
**Story Points:** 5

### 2. Patient User Story
**Title:** Book Appointment  
**Criteria:** As a Patient, I want to book an appointment with a specific doctor on a selected date.  
**Priority:** High  
**Story Points:** 8

### 3. Admin User Story
**Title:** Manage Doctor Profiles  
**Criteria:** As an Admin, I want to add or remove doctors from the system.  
**Priority:** Medium  
**Story Points:** 3
